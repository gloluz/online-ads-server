# Online ads server project

## Description

This project is an API to manage online ads.

## Installation

First, clone the project :

```
git clone git@github.com:gloluz/online-ads-server.git
```

Then you must install the project :

```
npm ci
```

## Comands

### Start the server

```
npm start
```
